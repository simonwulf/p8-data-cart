# p8-cart

Reusable cart functionality for JavaScript/TypeScript projects.

## Installation

```sh
npm install p8-cart
```

## Usage

Import and use the library in your project:

```js
import { /* exported functions/classes */ } from 'p8-cart';
```

## Development

- Source code is in `src/`
- Entry point is `src/index.ts`
